The project BEingenious contais three folders.
Assets contain all the javascript, css files for user interface design.
Scripts contain all the python files for the backend of the website.
The templates folder contains all the html files for building the webpages.

How to run:
1)Google App SDK should be installed on the system.
2) Once it is installed, the server can be run on local host by running the following command,(This command should be run from inside the folder):

python "C:\Program Files (x86)\Google\Cloud SDK\google-cloud-sdk\bin\dev_appserver.py" app.yaml 

3) Now the website is launched on localhost:8080

4) The website has also been deployed on be-ingenious.appspot.com and can be accesses from anywhere.
